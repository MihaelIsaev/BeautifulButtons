Beautiful Buttons
================

Images downloaded from http://nathanbarry.com/designing-buttons-ios5/

For ios 5+ only!

ARC only!

For example:
```
//in your .h file
#import "BeautifulButtons.h"

//in your .m file

@synthesize myButton;

-(void)viewDidLoad
{
  [BeautifulButtons makeButton:self.myButton styled:BBStyleOrange];
}
```
Enjoy!
